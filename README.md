#  Escena del Crimen - Videoclub Full Stack
###  Este proyecto es una aplicación web completa (Full Stack) diseñada para gestionar un catálogo de películas de cine negro y thrillers. Permite la visualización, registro y administración de títulos mediante una arquitectura moderna.
##  🛠️ Tecnologías Utilizadas
##  Frontend
###  📌React.js: Biblioteca principal para la interfaz de usuario.
###  📌CSS3: Diseño personalizado y responsive (Carruseles, Cards y Modales).
###  📌Vite: Herramienta de construcción rápida.

##  Backend
###  📌Java 17: Lenguaje de programación del servidor.
###  📌Spring Boot: Framework para la creación de la API REST.
###  📌Hibernate / JPA: Para el mapeo y gestión de datos.

##  Base de Datos
###  📌MySQL: Motor de base de datos relacional para el almacenamiento persistente.

##  🚀 Arquitectura del Proyecto
###  El proyecto sigue un flujo de datos circular que garantiza la integridad de la información:

###  💡Frontend (React): Captura los datos a través de formularios dinámicos.
###  💡API (Spring Boot): Procesa las peticiones HTTP y gestiona la lógica de negocio.
###  💡Database (MySQL): Almacena de forma estructurada las películas y categorías.

##  📋 Funcionalidades Principales
###  ✅Catálogo Dinámico: Visualización de películas agrupadas por subgéneros (Cine Negro, Mafias, Thriller).
###  ✅Gestión de Películas (CRUD): Formulario completo para agregar nuevos títulos a la base de datos en tiempo real.
###  ✅Integración de Multimedia: Soporte para pósters y trailers de YouTube.
###  ✅Diseño Temático: Interfaz inmersiva adaptada al género del cine policial.

##  ⚙️ Instalación y Configuración
###  💡Clonar el repositorio: git clone
###  💡Configurar la Base de Datos:  Importar el archivo .sql incluido o ejecutar el script de creación de tablas en MySQL Workbench.
###  💡Ejecutar el Backend:  Abrir el proyecto en IntelliJ IDEA y ejecutar la clase principal de Spring Boot.
###  💡Ejecutar el Frontend:  ***cd frontend -npm install  -npm run dev***
